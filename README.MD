# Проект "Интернет-магазин"

## Описание:

Проект "Интернет-магазин" - это приложение на Python, 
которое выполняет функции электронной торговли.


## Установка:

+ Установите на свой компьютер GitHub.
+ Клонируйте репозиторий с GitHub с помощью веб-URL.
```
 git clone git@github.com:den-solodovnikov/HomeWork_Mod_4.git
```


## Использование:

В текущем виде, программа запускается в модуле main.py:
Для проверки работы функций в модуле main.py


## Примеры использования функций:

+ Модуль product содержит функцию инициализации класса Product

+ Модуль category содержит функцию инициализации класса Category

+ Модуль utils
содержит функции подгрузки данных из JSON-файла:

**read_json(path) -> dict:**
Открывает файл JSON и считывает данные в словарь

**create_objects_from_json(data: dict) -> list:**
Создает объекты классов Product и Category из данныхб полученных из внешнего файла.
